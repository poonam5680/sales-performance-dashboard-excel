# Excel Sales Dashboard

## Project Overview

This project is an interactive Excel Sales Dashboard built to analyze sales performance across multiple cities.

The dashboard provides quick insights into:
- Total sales by sales executive
- Target hit percentage
- Performance gap from target
- City-wise filtering
- Trend comparison through charts

---

## Features

### City-wise Filters
Users can filter sales data by:
- Chennai
- Delhi
- Mumbai
- Nagpur
- Patna
- Pune
- Ranchi
- Surat

### KPI Metrics
Dashboard tracks:
- Total Sales
- Target Hit %
- Away From Target %

### Visualizations
- Bar Chart
- Pie Chart
- Line Chart

### Data Source
Raw sales data maintained in separate sheet:
- Raw Data
